资源文件夹

详细介绍
如果你的Hexo项目中只有少量图片，那最简单的方法就是将它们放在 source/images 文件夹中。然后通过类似于

的方法访问它们。
想要更有规律地提供图片和其他资源以及想要将他们的资源分布在各个文章上，config.yml 文件中的 post_asset_folder 选项设为 true 来打开。

当资源文件管理功能打开后，Hexo将会在你每一次通过 hexo new [layout] <title> 命令创建新文章时自动创建一个文件夹。
这个资源文件夹将会有与这个 markdown 文件一样的名字。将所有与你的文章有关的资源放在这个关联文件夹中之后，你可以通过相对路径来引用它们，这样你就得到了一个更简单而且方便得多的工作流。

资源相对路径引用的标签插件
